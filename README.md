# 2021IndieGameAcademy
--------------

## 개발 환경
- 사용 언어: C#
- 게임 개발 프레임워크: Unity
    - Unity 버전: 2020.3.24f1 

## 개발 프로젝트 내용
- 제목: 마계인천
- 장르: 텍스트 턴제 액션 어드벤처
- 플랫폼:모바일(안드로이드)
- 플레이 목적: 전략적으로 주사위를 굴려 적을 물리치고, 선택지르 골라 다양한 방향으 뻗어나가는 스토리 감상
- 게임의 차별점: '인천'이라는 현실적인 지역과 비현실적인 판타지 세계의 조화로우 조합
- 개발 기간: 12/17 - 12/18 

## 유료 에셋
- Dark - Complete Horror UI: UI르 위한 에셋
- Dice 2.5D: 주사위 에셋
